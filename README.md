# emacs-dotfiles

Emacs Dotfiles

## Tutorial

## Org

* `C-c '` 打开一个主模式的（python or elisp） 来编辑代码 结束也是一样

## Command

### M-x

* `M-x shell`
* `M-x eval-buffer`
* `M-x customize`     自定义界面设置
* `M-x customize-themes`
* `M-x list-packages`
* `M-x term`          调出原生 Terminal
* `M-x ansi-term`     这个用的更多 更快，而且可以用 emacs 快捷键
